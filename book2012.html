前言/Preface

2008年6月的某一天，幸运的收到了人民邮电出版社杨璐编辑的一封邮件。邮件是她的一本Flash方面书籍的写书邀请。

得到杨璐的邀请让我非常开心。我在2004年也写过一本Flash动画的书:《网页设计师必读——网页动画Flash CS3》（2007年底由电子工业出版社出版）。遗憾的是，这本书通篇都是关于Flash的动画制作方面的，如动画贺卡、Flash MTV等，很少涉及代码。而接下来几年我从事的都是Flash互动设计方面的项目，特别是和朋友一起在4A、国内外设计公司做了大量ActionScript方面的Flash互动网站开发。在这期间，很希望能够再写一本关于Flash建站方面的书，以此整理自己在项目过程中的经验和收获。

杨璐的想法与我不谋而合。在她看来，无论是全站还是局部应用，在很多企业网站中都需要用到Flash，相关的实际应用是千变万化的。现在市场上还很少有书从Flash网站元素入手，逐步深入到Flash全站，我们的书因此应是很有价值的。我们仔细商量了下书的主题和层次结构。杨璐勾画了本书的大框架，我将细致的目录充实进去，并花了一个月的时间制作所有的样例（见样例）。一切准备就绪之后，就开始了书稿的写作。

这本书从7月中旬开始，边写边和杨璐讨论，修改，前前后后一共花了三个半月。暑假期间在西昌，几乎天天窝在家里写，进度还是比较快的，半个月不到就完成了前八章的初稿。杨璐计划10月全部交稿，来年1月旺季出书。于是从9月中旬开始写书的后半部分，一直到10月的最后一天，总算把所有书稿都完成了。

然而，接下来却发生了意想不到的事情。由于08年开始，计算机图书市场开始变得严重不景气，导致出版社调整了出版计划，计算机图书的出版大幅缩水，我们的书也被排在了计划以外。09年，我们的书没有能出版，然后是2010，2011。。。

时间一晃就到了2012。在这几年漫长的时间中，Flash本身在不断更新和升级中。2010年Flash CS5推出，2012年Flash CS6推出 ，其间ActionScript 3.0也有了很多更新。我们只能继续动态修改书稿，希望书稿能够体现最新的技术更新。很多时候这是比较大的修改，大部分的图例都重新制作过2到3次。 漫长的等待，本来我都对这本书的出版失去希望了。直到今年，杨璐告诉我这书已经进了流程，总算可以出版的时候，心里真是说不出的喜悦！

出版的过程也不是一帆风顺，原定7月底上市，结果因为种种原因进展比较缓慢。甚至在印刷阶段，还因为排版公司选择的书眉字体出现问题，书不得不全部重印，增加了不少成本（痛）。为此杨璐也非常郁闷，她说，这也许就是“好事多磨”吧!

今时今日，Flash的热度已不如从前。CSS、JavaScript，特别是HTML5给Flash造成了很大的冲击，移动应用开发成为时下热门。但不可否认，Flash仍然是一个伟大的软件。不管您是专业的程序开发者，还是毫无编程功底的美术设计师，Flash都是每个人展示自己才华的舞台。

介绍/About

书名：《Flash ActionScript 3.0 全站互动设计 》(附光盘)

作者：刘欢

责任编辑：杨璐

ISBN: 9787115287953

出版时间：2012-10-1

页　　数：488

价　　格: ￥ 69.80

 

图书描述： 《Flash ActionScript 3.0全站互动设计》从ActionScript 3.0的基础语法入手，以当下Flash网站常见功能模块和互动元素为重点，讲解ActionScript 3.0在互动网站中的应用，包括音乐播放器、视频播放器、下拉菜单、滚动条、图片轮动、网页相册、浮动导航、XML折叠菜单、数据筛选器、新闻显示器、RSS阅读器和3D旋转Logo等，并精选两个完整网站，剖析全站互动设计细节，让读者从新手成为网站互动设计高手。
　　 《Flash ActionScript 3.0全站互动设计》配套光盘收录ActionScript 3.0完全实用视频手册，弥补本书篇幅所限，全面讲解了ActionScript 3.0应用的方法与技巧，更收录了本书所有案例的制作过程教学视频，帮助读者更好地理解和掌握关键技术。

 

本书简介：本书分三大部分：第一部分ActionScript编程篇，第二部分互动元素设计篇，第三部分Flash全站设计篇（见目录）。

严格地说，本书并不是一本ActionScript 3.0的大百科类书籍，并没有像参考书那样，将ActionScript 3.0中的每一个对象的属性、方法和事件都罗列和展示出来，因为对于对于日常的Flash开发来说，不是所有的功能和方法都是常用的，将它们全部搬到书上会让读者很难分辨阅读重点，让人头晕脑胀，过目即忘，其效果甚至不如直接阅读Flash自带的参考手册。反之，本书主要介绍了互动开发中常用的ActionScript知识，并且着重通过互动实例对这些代码进行灵活运用，以此使读者深入学习和掌握ActionScript 3.0。

本书还包括了一张视频学习光盘，里面有约10小时的视频实例解说，涵盖了一些书中限于篇幅没有详细指出的开发经验，特别对于初学者来说是很有价值的学习资料。

希望在读完本书后，即使是毫无基础的读者，也能熟悉掌握ActionScript 3.0，同时对ActionScript 3.0的编程和Flash在互动设计中的应用有更加直观的认识。本人学识和水平有限，书中难免有错误和遗漏，请各位不吝指正。我的电子邮箱是：liuhuan[-At-]liuhuan.com。

勘误/Errata

  1. Page 30:var i:int在未赋值状况下，trace输出结果是0而非undefined。（谢谢网友☆随风☆指出）
  2. Page 31:倒数第7行：
    原文：var word:String = '”this is an apple!””;//错误
    应为： var word:String = ””this is an apple!””;//错误
  3. Page 41:“date”类打成了“data”，应为“date”。（谢谢网友☆随风☆指出）
  4. Page 55:两处“如图2.30所示”都应为“如图2.31所示”。
  5. Page 63:
    原文“设置candle元件的mouseChildren属性为flase”
    应为“设置candle元件的mouseChildren属性为false”
  6. Page 64：注释“听对象为舞台本身”应为“侦听对象为舞台本身”。
  7. Page 80：原文“fireSystem = new Missile ();” 应为“weapon = new Missile ();”
  8. Page 141：FileReference的upload方法没有讲仔细，可能会让读者比较困扰，这里再解释一下：asp页面里创建上传文件对象的代码为“set file=upload.file("Filedata")”，其中的参数使用了Filedata，这是因为FileReference向后台所发送数据的name属性在默认情况下即为 "Filedata"。
  9. Page 235：子标题“县浮”应为“悬浮”。
  10. Page 360, Page 365：import fl.transitions.Tween 和 import fl.transitions.easing.*在本例中没有任何作用，应去掉。
  11. Page 458, 页尾类名和基类名设置有误，设置mMenuItem的类名应为com.site.ui.MenuItem，基类不用设置。
  12. Page 480：mDetail.visible=false 原注释为“隐藏mList元件”，应为“隐藏mDetail元件”。
  13. 关于ProLoader：这是Flash CS5.5中提出的一种代替Loader的对象。Flash在参考手册中明确提出：“当加载由 Flash Professional 生成的 SWF 文件时，应使用 fl.display.ProLoader 而非 flash.display.Loader” 。然而事实上 ProLoader主要在类似加载使用了TLF文本的SWF等情况下有别于Loader(即 “对于使用 RSL 预加载的 SWF 文件尤其有用” )，这些都是在一般项目中较为少见的情形。而一旦使用ProLoader，必须使用Flash CS5.5以上版本才能正确发布文件，在团队协作中，如果还有成员使用之前的版本就会产生问题。在没有太大区别的情况下，我个人仍然建议继续使用Loader。本书的相关案例也都使用Loader而非ProLoader。
  14. 光盘 9.循环语句：这一节录制的时候因为紧张，在for循环输出i，j值的时候说错了，问题出在我在连接的时候用&号了，应该为+号，非常抱歉。

后记/Postscript

书已出版一个月了，重新读了一遍，又看了一下光盘，发现书和光盘里还有很多的纰漏和错误，一些知识点并没有讲的很透彻，这些都让我如芒在背，羞愧不已，也有负编辑杨璐的重托。果然是本人水平有限。

有一部分书中的问题我已经更正在勘误中了，估计还有更多的错误和缺漏，在此就这些问题和错误向大家表示深深的歉意。

限于本书篇幅（策划本书时规划总页数在360页以内，现在已经远远超了），有不少东西也没法摊开来写的很详细，其实本书的内容只是商业互动设计的一些较基础的知识，希望能够籍此帮助初学者和中级开发人员快速起步，因此我的目的是尽量做到浅显易懂，很惭愧本书也许并没有做到这点。事实上，一个成熟的商业项目，涉及的知识点和难点会更多，而且在开发时间上也有非常严格的要求，需要开发者在极短的时间内找到合适的解决方案。如果有机会的话，也许还能写一些更高阶的案例，讲讲更复杂的情况。但从flash目前的发展来看，可能这样的机会会比较渺茫了。

此外，我觉得光盘视频是一种非常好的学习工具，也许能够帮助您更快速学习到一些知识。本书的附录光盘中是我录制的大约50段视频，虽然本人普通话非常不标准，而且因为录制的时候较为紧张，还有一些说错的地方，但您还是可以随便浏览看看，也许要比直接读书更加轻松一些。我上传了一些视频在youku上面，没有购买本书的朋友也可以在这里观看。

最后，我要向每一位购买本书的朋友道一声：谢谢！感谢你们的支持，感谢你们对我的肯定和帮助！
